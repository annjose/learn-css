Pseudo Classes

Example of first-child pseudo class. All italics are green by default. If the first child is bold, it will be red. If the first child is italic, it will be blue.
First child is bold, hence it should be red. | First italic child, but not the first child in the parent (hence it should be the default green) | Second child bold (should be black) | Second child italic (should be green)
First child is italic, hence it should be blue. | This is the second child (a bold), but not the first child in the parent (hence it should be the default black) | Second child bold (should be the default black) | Second child italic (should be the default green)
Example of first-of-type pseudo class All italics are green by default. Any first bold type will be red. Any first italic child is italic, it will be blue.
First bold child (should be red) | First italic child, but not the first child in the parent (should be blue). | Second bold child (should be the default black) | Second italic child (should be the default green)
